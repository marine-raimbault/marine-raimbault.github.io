# Create a Blog in 5 Minutes

You too can create a blog in just 5 minutes! The fastest way is to [install Jekyll](https://jekyllrb.com/docs/) and run the command:

```bash
jekyll serve
```

## Free Hosting with GitHub Pages

You can then host your blog for free through GitHub Pages. The process is described succinctly in the GitHub Pages [Quickstart tutorial](https://docs.github.com/en/pages/quickstart).
    
## Quick Steps

1. Install Jekyll following the [official documentation](https://jekyllrb.com/docs/)
2. Create your blog with `jekyll serve`
3. Host it for free using [GitHub Pages](https://docs.github.com/en/pages/quickstart)
